# Bootstrap Email

This project is a simple HTML email template built using Bootstrap or Bootstrap Email styles. It was created for practice as part of the Bay Valley Tech Code Academy curriculum.

## Project Files

1. index.html  
   Contains the HTML for the email template.

## How to View the Project

1. Download or clone the repository.
2. Open the index.html file in a browser to see the email template.

No build tools or additional setup are required.

## What This Project Includes

1. A basic email layout written in HTML
2. Bootstrap or Bootstrap Email based styling
3. Static content only with no dynamic behavior

## Purpose

This project was built as part of the final work for one of the modules at Bay Valley Tech Code Academy. It shows how to structure and style an HTML email using familiar CSS classes and simple layout techniques.
